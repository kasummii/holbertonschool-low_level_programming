# 0. 98 Battery st.

## Description
This project implements a function that takes a pointer to an int as parameter and updates the value it points to to 98.

## Files
- 0-reset_to_98.c: contains reset_to_98 function
- main.h: header file with prototype for reset_to_98

## Compilation
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 0-main.c 0-reset_to_98.c -o 0-98

## Usage
Run the program to see output before and after calling reset_to_98.

## Allowed functions
write, printf, putchar, puts, malloc, free
